# Notes Book App

This project provides an API for managing notes. 



### Installation

1. Clone the repository:

```bash
git clone https://github.com/your-repository/notes-api.git
cd notes-book-app

```

### API Documentation

API documentation is available at http://localhost:3000/api-docs.
